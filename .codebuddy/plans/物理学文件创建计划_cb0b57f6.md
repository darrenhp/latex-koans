## 产品概述

在 latex-koans 项目中新增物理学章节，覆盖从初中物理到前沿物理理论的完整内容体系。

## 核心功能

- 创建物理学.md 文件：精简版物理学概念与公式汇总，按分支组织
- 创建 chapters/12-物理学.yaml 文件：50-100 道 LaTeX 物理公式练习题
- 更新 chapters/index.yaml：添加物理学章节入口
- 内容覆盖：力学、热学、电磁学、光学、近代物理、量子力学、相对论、粒子物理（含宇称不守恒、爱因斯坦引力场方程、弗里德曼方程、标准模型拉格朗日量）

## Tech Stack Selection

- 文件格式：Markdown (.md) + YAML (.yaml)
- 公式语法：LaTeX（通过项目已有的 MathJax 3 渲染）
- 编辑工具：直接文件创建，无需额外依赖

## Implementation Approach

采用静态文件添加策略，严格遵循项目现有格式规范：

1. **物理学.md**：参考数论.md 的精简风格，按物理学八大分支组织，每个概念仅包含定义描述和对应 LaTeX 公式，不做详细推导
2. **chapters/12-物理学.yaml**：遵循现有 koans 数组格式，每项包含 title/tag/difficulty/latex/description 五个字段
3. **chapters/index.yaml**：在现有章节列表末尾追加物理学章节条目

内容分级策略：

- easy：基础公式（牛顿定律、欧姆定律、热力学定律等）
- medium：进阶公式（麦克斯韦方程组、薛定谔方程、洛伦兹变换等）
- hard：前沿理论公式（爱因斯坦场方程、标准模型拉格朗日量、弗里德曼方程等）

## Implementation Notes

- YAML 中转义：LaTeX 的反斜杠需双写（如 `\\frac` 而非 `\frac`），冒号后需空格
- 描述字段可包含 LaTeX 公式，使用 `$... 或 `$... 包裹
- chapter 字段统一为"物理学"，与 index.yaml 中注册的章节名称一致
- 题目 id 在 YAML 中不需要（由 index.html 的加载逻辑自动生成）

## Architecture Design

本任务为纯静态文件添加，不涉及架构变更。新增文件与现有项目结构完全兼容，index.html 的动态加载逻辑无需修改。

## Key Code Structures

YAML 题目条目格式示例：

```
koans:
  - title: 牛顿第二定律
    tag: 力学
    difficulty: easy
    latex: 'F = ma'
    description: '牛顿第二定律：$F = ma$，力等于质量乘以加速度。'

  - title: 爱因斯坦引力场方程
    tag: 相对论
    difficulty: hard
    latex: 'R_{\mu\nu} - \frac{1}{2}Rg_{\mu\nu} + \Lambda g_{\mu\nu} = \frac{8\pi G}{c^4}T_{\mu\nu}'
    description: '爱因斯坦引力场方程：$$R_{\mu\nu} - \frac{1}{2}Rg_{\mu\nu} + \Lambda g_{\mu\nu} = \frac{8\pi G}{c^4}T_{\mu\nu}$$'
```
